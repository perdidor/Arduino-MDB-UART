Sources are Atmel (Microchip) Studio 7.0 solution with 2 firmware projects:
MDB-UART-MASTER (firmware images in this repository compiled from it)
MDB-UART-SLAVE (emulating cashless devices and Universal Satellite Devices. Project is not finished yet: 90% code written but not tested with real MDB POS terminal.)
